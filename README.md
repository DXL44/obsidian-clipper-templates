# obsidian-clipper-templates
My personal templates for Obsidian Web Clipper. Free to use, but might require some tweaking for your vault.

# Explanation

[Obsidian Web Clipper](https://obsidian.md/clipper) is a browser extension that allows you to save content of a web page to an Obsidian vault in one click. 

## Folders
- The "templates" folder has templates that are geared towards general use. You may want to edit them to fit your vault better, but they're a good starting point. 
- The "templates-personal" folder has unedited templates that are geared to work in my personal vault. Unlike the templates above, there's a lot more you'll need to edit out i.e. links to pages in my vault or certain frontmatter properties I use.
